# Sigils

## Usage

Install the cli

```
[sudo] npm i -g sigils
```

Describe your intent and receive an output

```
sigils 'this my wish to obtain the strength of a tiger'
```

or use as a module

```
[sudo] npm i sigils
```

```js
const sigils = require('sigils');

sigils.sigil('this my with to obtain the strength of a tiger');
```
